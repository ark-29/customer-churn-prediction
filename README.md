# Customer Churn Prediction Using Random Forest

## Overview

This project predicts whether a customer is likely to churn using machine learning techniques on the IBM Telco Customer Churn dataset.

## Features

* Data preprocessing
* Feature engineering
* One-hot encoding
* Random Forest classification
* Model evaluation
* Customer churn prediction

## Technologies Used

* Python
* Pandas
* Scikit-learn
* NumPy
* 
## Model Performance

* Accuracy: 79%
* Evaluated using precision, recall, and F1-score

## Dataset

IBM Telco Customer Churn Dataset
